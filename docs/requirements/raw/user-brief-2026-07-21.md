ผมต้องการทำเว็บชื่อ Deckoala โดยมี DNS คือ deckoala.dimenshade.com
เว็บไซต์นี้ผมต้องการให้สามารถนำไป self-host ได้ง่าย ๆ โดยที่อาจจะเป็น compose.yml สัก 1 ไฟล์
แล้วก็ในหน้าเว็บนั้นโดยเว็บไซต์นี้ Markdown presentation tools
โดยในหน้าเว็บไซต์จะอาจจะเขียน Markdown, LaTex หรือ Drag & Drop เพื่อสร้างสไลด์ได้
ขณะสร้างสไลด์ก็สามารถ Preview ไปพร้อม ๆ กันได้
สามารถติดตั้ง font เอง รวมถึงดึง font จากแหล่งต่าง ๆ มาติดตั้งง่าย ๆ
แล้วก็สามารถใช้งานได้ในทุกอุปกรณ์แบบสะดวกและใช้งานง่าย
สามารถกด Present ผ่านเว็บ หรือ Export เป็น PDF ได้
โดยอยากให้สีพื้นหลัง Default เป็นสี #F8F8FF ส่วนสีตัวอักษรคือ #0B1215
